# Көчмөн Ордо (Kochmon Ordo)

Статический веб‑проект на HTML/CSS/JS для витрины и входа в кыргызскую игровую платформу. Сайт включает главную страницу с авторизацией/регистрацией, профиль пользователя, страницу игр с переключателем языка, а также разделы “О нас” и “Контакты”.

## Возможности
- Регистрация и вход без бэкенда (данные хранятся в `localStorage`).
- Профиль пользователя со статистикой, достижениями и прогрессом.
- Настройки: язык интерфейса, тема оформления, уведомления.
- Меню выбора игр и открытие внешних игр в новой вкладке.
- Автообновление интерфейса при возврате со вкладки игры.
- Валидация email/пароля/имени пользователя и индикатор силы пароля.

## Страницы
- `index.html` — главная страница, авторизация/регистрация, профиль, CTA‑секция.
- `games.html` — каталог игр с внешними ссылками и переключателем языка.
- `about.html` — “О нас”.
- `contacts.html` — контакты и email поддержки.

## Игры (внешние ссылки)
- Көчмөн Орноку
- Канаттуу куш (Flappy Bird)
- Кыз куумай (IEI)
- Кыз куумай (DGONI)
- TynchysPy
- Rich Wheel
- Kyrgyz Memory

Ссылки на игры находятся в `games.html` и в `mega.js` (функция `startGame`).

## Локализация
- `games.html` использует `games-i18n.js` и локальные словари для `ky`, `ru`, `en`.
- Главная страница управляется `LanguageManager` в `mega.js`. Сейчас переводы `ru/en` наследуют киргизские строки (можно расширить при необходимости).

## Хранение данных
Проект не использует сервер. Все данные сохраняются в `localStorage`:
- Префикс: `kochmonOrdo_` (и миграция с `ndnStore_`).
- Сессии, пользователи, настройки и временные данные сохраняются в `TempStorage`.
- Язык страницы игр хранится в `ndn_game_lang_v1`.

Важно: пароль хешируется простой демонстрационной функцией (не для продакшена).

## Технологии и зависимости
- Чистый HTML/CSS/JS.
- Внешние CDN:
  - Font Awesome (иконки).
  - Google Fonts (в `games.html`).

## Структура проекта
- `index.html` — основная логика интерфейса.
- `mega.js` — поведение UI, авторизация, профиль, настройки, валидации.
- `mega.css` — стили главной страницы и профиля.
- `games.html` — витрина игр.
- `games-i18n.js` — переключение языка на странице игр.
- `games-theme.css` — стили страницы игр.
- `about.html`, `contacts.html` — информационные страницы.

## Как запустить локально
1. Откройте `index.html` в браузере.
2. Для каталога игр откройте `games.html`.

Сборка и сервер не требуются.

## Примечания
- Все данные локальные и привязаны к браузеру пользователя.
- Для корректного отображения киргизских текстов файлы должны быть в UTF‑8.
